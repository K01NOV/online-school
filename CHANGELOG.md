# Список изменений

## [0.9.0-beta] - 2026-02-22
### Добавлено
- Полностью переписана архитектура на MVC.
- Внедрена поддержка `.env` файлов и скрытие токенов API.
- Подключено API Яндекс.Диска для работы с учебными материалами.
- Реализована регистрация и авторизация.
- Создана структура каталога (Классы -> Предметы -> Темы).
- Добавлен роутинг

### В процессе
- Улучшение алгоритма поиска по сайту.
- Заполнение шаблонов уроков реальным контентом.

## [0.9.1-beta] - 2026-02-25
### Добавлено
- Система токенов для форм
- Роутер генерирует защитные токены
- Внедрена защита от CSRF
- Базовый интерфейс админ панели
- Суб роутер для админ панели
- Рефакторинг регистрации: исправление ошибки с enum ролей пользователей, добавлена поддержка en/ru для enum

### В процессе
- Разработка CRUD раздела админ панели
