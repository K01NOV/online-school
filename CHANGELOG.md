# Список изменений

## [0.9.0-beta] - 2026-02-22
### Добавлено
- Полностью переписана архитектура на MVC.
- Внедрена поддержка `.env` файлов и скрытие токенов API.
- Подключено API Яндекс.Диска для работы с учебными материалами.
- Реализована регистрация и авторизация.
- Создана структура каталога (Классы -> Предметы -> Темы).
- Добавлен роутинг

### В процессе
- Улучшение алгоритма поиска по сайту.
- Заполнение шаблонов уроков реальным контентом.

## [0.9.1-beta] - 2026-02-25
### Добавлено
- Система токенов для форм
- Роутер генерирует защитные токены
- Внедрена защита от CSRF
- Базовый интерфейс админ панели
- Суб роутер для админ панели
- Рефакторинг регистрации: исправление ошибки с enum ролей пользователей, добавлена поддержка en/ru для enum

### В процессе
- Разработка CRUD раздела админ панели

## [0.9.2-beta] - 2026-02-27
### Добавлено
- Возможность просмотра таблиц в админ панели
- Динамическая верстка админ панели
- Автоматическое определение типов данных колонок таблиц
- Методы безопасной вставки и удаления записей

### В процессе
- Добавление аякс для возможности изменять данные таблиц без обновления страницы

## [0.9.3-beta] - 2026-03-10
### Добавлено
- Внедрена система роутинга для API
- Реализована логика обработки CRUD-операций (Create, Read, Update, Delete) в TableApi.php
- Добавлена функция update_at в CrudModel
- Реализована поддержка динамического управления таблицами через AJAX-запросы
- Внедрена система валидации CSRF-токенов в виде отдельного класса с функциями генерации токена и его валидации
- Настроено разграничение прав доступа: автоматическое перенаправление администраторов в админ панель
- Добавлен favicon проекта
- Обновлены мета-теги
- Реализована серверная сортировка результатов поиска
- Удалены устаревшие функции сортировки в script.js
- Внедрен новый js скрипт для обработки API-запросов

### Исправлено
- Исправлены некорректные ссылки в шапке сайта
